# 小程序版个人博客

为了试水小程序而制作的我的[个人博客](https://seewang.me)小程序版，实现了内容获取，基础样式，页面路由等简单功能。

### 工具/库
- FontAwesome: 图标
- LeanCloud: 内容储存
- wxParse: markdown解析
